About

This randomizer helps your squad quickly assign unique VALORANT agents while honoring optional role priorities. This project was programmed using Python and Django, with styling assets stored using AWS S3, deployed on Render, and backed by a PostgreSQL database.


This project is not affiliated with or endorsed by Riot Games and is not an official VALORANT randomizer site. It is a fan-made passion project created by a VALORANT player.
