About

This randomizer helps you and your friends quickly assign VALORANT agents while having the option to prioritize roles. This project was programmed using Python and Django, with styling assets stored using AWS S3, deployed on Render, and backed by a PostgreSQL database.


This project is not affiliated with or endorsed by Riot Games and is not an official VALORANT randomizer site. It is a fan-made passion project created by a VALORANT player.
